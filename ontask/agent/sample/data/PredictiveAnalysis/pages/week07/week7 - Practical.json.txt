<h3>Practical</h3>
<h4>Overview</h4>
<p id="yui_3_17_2_1_1691839755294_129"><span>In this Practical we will learn how to produce a set of classification rules using the RIPPER algorithm.</span></p>
<h4>Objectives</h4>
<ul>
    <li><span lang="EN-US">Understand the principles of the RIPPER algorithm.</span></li>
    <li><span lang="EN-US">Learn classification rules on a sample dataset.</span></li>
    <li><span lang="EN-US">Apply the learned rules on test tuples.</span></li>
</ul>
