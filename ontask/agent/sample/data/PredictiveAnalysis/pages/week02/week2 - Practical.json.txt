<p>In this Practical we will introduce some of the commonly used data exploration techniques in R. We will focus on several useful R commands which allow you to observe data and prepare descriptive statistics.</p>
<h5><strong>Objectives</strong>:</h5>
<ul>
    <li>Load and explore data</li>
    <li><span>Create descriptive statistics</span></li>
    <li>Observe data using univariate plots</li>
</ul>
