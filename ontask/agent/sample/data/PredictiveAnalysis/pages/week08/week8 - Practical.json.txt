<h3>Practical</h3>
<h4>Overview</h4>
<p><span lang="EN-US">In this Practical we will revisit rule-based classifiers and learn how to build and interpret k-nearest neighbor (KNN) and Na&iuml;ve Bayes classifiers.</span></p>
<p><span lang="EN-US">We will learn about several useful packages (some being used before) and introduce a method for finding an optimal KNN model.</span></p>
<h4>Objectives</h4>
<ul>
    <li><span lang="EN-US">Build a predictive model using KNN algorithm.</span></li>
    <li><span lang="EN-US">Choose optimal<i><span>&nbsp;</span>K<span>&nbsp;</span></i>for a KNN model.</span></li>
    <li><span lang="EN-US">Build a Na&iuml;ve Bayes classification model.</span></li>
    <li><span lang="EN-US">Build a rule-based classification predictive model.</span></li>
    <li><span lang="EN-US">Make a prediction using models we built.</span></li>
</ul>
